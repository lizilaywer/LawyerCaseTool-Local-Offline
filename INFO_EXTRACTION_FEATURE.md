# 信息识别功能开发文档

## 功能概述

已在律师案卷自动化生成工具中添加"信息识别"功能，支持从身份证、户口簿、护照、驾驶证、营业执照、判决书、裁定书等图片或PDF文件中自动提取信息，并填充到案卷模板变量中。

## 更新历史

### v1.2.1 (2026-03-11) - 字段全面应用优化

**优化内容**:
- 点击"应用到案卷变量"后，所有识别字段都会应用到案卷表单
- 已存在的变量直接填充值
- 不存在的变量自动创建并添加到模板配置
- 自动为新变量生成中文标签

**涉及文件**:
- `src/core/ocr/field_matcher.py` - 新增 `match_all()` 方法
- `src/gui/info_extraction_dialog.py` - 优化提示信息
- `src/gui/main_window.py` - 支持动态创建新变量
- `src/gui/widgets/variable_input.py` - 支持动态添加控件

## 使用流程

1. **打开信息识别**
   - 在主界面点击"信息识别"按钮
   - 或从菜单栏进入

2. **添加文件**
   - 点击"+ 添加图片/PDF"按钮
   - 或拖拽文件到左侧列表
   - 支持批量添加

3. **开始识别**
   - 点击"开始识别"按钮
   - 等待识别完成（显示进度条）
   - 系统自动判断文档类型并提取字段

4. **核对结果**
   - 低置信度字段（<80%）会自动标红
   - 点击字段可直接编辑修正
   - 置信度显示在字段右侧

5. **应用数据**
   - 点击"应用到案卷变量"自动填充当前表单
   - 或点击"保存并应用到案卷"保存记录并填充
   - **系统会自动匹配识别字段与模板变量（v1.2.1+）**
   - **所有识别到的字段都会被应用**：已存在的变量直接填充，不存在的自动创建新变量
   - 弹窗会显示匹配详情：匹配现有变量数 + 创建新变量数

6. **导出数据**（可选）
   - 支持导出为 JSON、Excel、Word 格式
   - 方便在其他地方使用

## 注意事项

1. **首次使用 PaddleOCR**：
   - 首次运行时会自动下载 OCR 模型（约 100MB）
   - 需要联网下载，或手动下载模型文件放置到指定目录

2. **PDF 处理**：
   - PDF 识别依赖 PyMuPDF 库
   - Windows 用户可能需要安装 Visual C++ Redistributable

3. **识别准确率**：
   - 图片清晰度直接影响识别准确率
   - 建议提供 300 DPI 以上的扫描件
   - 倾斜图片会影响识别效果

4. **数据存储**：
   - 识别记录存储在 `%APPDATA%/LawyerCaseTool/extracted_info/`
   - 定期清理过期记录可释放磁盘空间
