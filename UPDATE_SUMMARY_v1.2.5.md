# 版本 1.2.5 更新摘要

**发布日期**: 2026-03-15

**版本号**: 1.2.5

---

## 概述

本次更新对律师案卷自动化生成工具进行了全面的 UI 现代化改造，大幅提升了界面的美观度和用户体验。同时新增了实用的搜索筛选功能和文件夹预览编辑功能。

---

## 主要更新内容

### 1. UI 全面现代化

#### 主窗口
- 现代化三栏式布局设计
- 统一的颜色系统和间距规范
- 全新的工具栏设计

#### 模板卡片
- 全新视觉设计：圆角、阴影、边框
- 类别标签改为按钮样式
- 悬停和选中状态优化
- 编辑按钮交互优化

#### 表单区域
- 更紧凑的输入框设计
- 统一的聚焦状态样式
- 改进的视觉层次

#### 文件夹树预览
- Emoji 图标系统（📁 📄）
- 扁平化选中状态
- 统计区域去边框化

### 2. 新增功能

#### 模板搜索
- 实时搜索过滤
- 支持名称和描述搜索

#### 分类筛选
- 5 个分类快速筛选按钮
- 清晰的选中状态指示

#### 文件夹预览编辑
- 拖拽排序功能
- 双击编辑名称
- 修改保存到模板

#### 应用图标
- 现代扁平化设计
- 25 个尺寸规格
- Windows ICO 和 macOS 格式支持

### 3. 问题修复

- 修复 QLineEdit 导入缺失
- 修复颜色键错误
- 修复字符串格式化问题
- 修复选中状态样式问题

- 水平边距: 16px
- 卡片圆角: 8px

- 设计尺寸: 1024x1024px
- 输出格式: PNG, ICO
- 尺寸范围: 16px - 1024px
