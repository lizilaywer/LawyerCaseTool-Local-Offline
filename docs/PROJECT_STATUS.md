## 电子化归档功能详情 (v1.3.0)

### 界面布局

```
┌─────────────────────────────────────────────────────────────────────────┐
│  工具栏：[导入变量] [导出变量]                    [选择归档模板] [替换导出] │
├──────────────┬──────────────────────────────┬────────────────────────────┤
│   左侧       │         中间                 │         右侧               │
│   变量定义   │       文件预览区             │      文件夹结构树          │
│   (280px)    │        (自适应)              │        (300px)             │
└──────────────┴──────────────────────────────┴────────────────────────────┘
```

### 功能特点

1. **变量定义面板**
   - 变量结构：名称 + 变量名 + 值输入框
   - 导入/导出 JSON
   - 与主界面模板系统同步（只增不减）

2. **文件预览区**
   - 支持 Word(.docx)、PDF、图片预览
   - 变量高亮显示
   - 右键菜单：设置为变量、定义为变量字段

3. **文件夹结构树**
   - 显示完整目录结构
   - 图标系统：📁📄🖼️
   - 单击重命名、双击打开、拖拽移动

4. **归档导出**
   - 默认模板路径：templates/
   - 自动识别模板变量
   - 历史记录保存
