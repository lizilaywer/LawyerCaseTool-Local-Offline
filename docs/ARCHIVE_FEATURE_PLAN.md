# 电子化归档功能实现计划

## 一、功能概述

### 1.1 功能目的

**核心场景**：律师手头已有整理好的案卷文件夹（包含各种 Word 文档、PDF、图片等），需要：
1. **浏览**现有案卷的文件夹结构
2. **提取**文件中的关键信息（当事人姓名、案号、日期等）
3. **生成**标准化的归档文档

**一句话概括**：从现有案卷材料中提取信息，填充到归档模板，一键导出规范文档。

### 1.2 核心流程

```
选择案卷文件夹 → 浏览文件结构 → 提取变量信息 → 选择归档模板 → 替换导出
```

### 4.3 左侧变量定义面板 (archive_variable_panel.py)

**类名**：`ArchiveVariablePanel`

**变量项结构**：
```
┌─────────────────────────────┐
│ 委托人姓名                  │  ← 变量名称（标签）
│ {{client_name}}             │  ← 变量名（灰色小字）
│ ┌─────────────────────────┐ │
│ │ 张三                    │ │  ← 值输入框
│ └─────────────────────────┘ │
│                           [×]│  ← 删除按钮
└─────────────────────────────┘
```

**功能列表**：

| 功能 | 方法 | 说明 |
|------|------|------|
| 加载变量 | `load_variables(variables: List[Dict])` | 从主界面模板系统加载 |
| 添加变量 | `add_variable(name: str, key: str, value: str)` | 添加新变量 |
| 删除变量 | `remove_variable(key: str)` | 删除指定变量 |
| 获取所有变量 | `get_all_variables() -> Dict[str, str]` | 返回变量键值对 |
| 导入 JSON | `import_from_json(file_path: Path)` | 从JSON导入 |
| 导出 JSON | `export_to_json(file_path: Path)` | 导出到JSON |
| 设置变量值 | `set_variable_value(key: str, value: str)` | 设置变量值 |
| 同步主界面 | `sync_from_main()` | 从主界面同步变量 |

**与主界面同步机制**：
- 初始加载时，从主界面模板系统获取所有变量
- 电子化归档中新增的变量 → 同步回主界面模板系统
- 主界面删除变量 → 电子化归档中对应变量不自动删除
- 只有用户手动删除才会移除变量

**继承**：`QWidget`

**右键菜单实现**：
- 设置为变量 → 弹出变量列表 → 选择后文字替换为 `{{变量名}}`
- 定义为变量字段 → 弹出变量列表 → 选择后自动填充左侧对应变量的值

**变量自动识别与同步**：
- 模板加载后自动识别所有 `{{xxx}}` 格式的变量
- 匹配左侧已有变量 → 关联显示
- 左侧没有的变量 → 自动新增到左侧变量列表

## 五、典型使用流程

```
1. 点击"电子化归档" → 选择案卷文件夹（如 "2024-001_张三_民间借贷"）
                          ↓
2. 左侧自动加载主界面模板系统的变量
   右侧显示文件夹结构
                          ↓
3. 双击打开"委托合同.docx" → 中间预览
                          ↓
4. 选中"张三" → 右键"定义为变量字段" → 选择"委托人姓名"
                          ↓
5. 左侧"委托人姓名"的值输入框自动填入"张三"
                          ↓
6. 继续浏览其他文件，提取更多变量值
                          ↓
7. 点击"选择归档模板" → 默认打开 templates/ → 选择"归档模板.docx"
                          ↓
8. 中间显示模板，自动识别变量并高亮
   模板中的新变量自动添加到左侧
                          ↓
9. 点击"替换导出" → 选择保存位置 → 生成最终归档文档
                          ↓
10. 变量记录自动保存到历史记录
```

## 八、错误处理

### 8.1 文件操作错误

- 文件夹不存在：提示用户重新选择
- 文件损坏：显示错误信息，跳过该文件
- 权限不足：提示用户以管理员身份运行

### 8.2 变量操作错误

- 变量名格式错误：提示正确格式
- 变量名重复：提示已存在
- 值验证失败：高亮显示错误字段

### 8.3 导出错误

- 模板文件不存在：提示重新选择
- 输出路径无效：提示选择有效路径
- 导出失败：显示详细错误信息
