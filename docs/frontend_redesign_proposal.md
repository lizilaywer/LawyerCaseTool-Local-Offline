# 律师案卷工具前端重设计方案

## 技术栈选型报告

### 为什么放弃 PySide6？

当前使用 PySide6 (Qt) 存在以下问题：
- **界面老旧**: Qt 的默认样式缺乏现代感，难以实现流畅的动画效果
- **开发效率低**: Python GUI 开发调试困难，热更新需要重启应用
- **样式定制难**: QSS 不如 CSS 灵活，复杂布局实现困难
- **Web 技术断层**: 无法利用现代 Web 生态的丰富组件库

## 核心技术详解

### 1. Tauri 2.0 (桌面应用框架)

**为什么选择 Tauri？**

| 特性 | Tauri 2.0 | Electron | Wails | PySide6 |
|------|-----------|----------|-------|---------|
| **包大小** | ~3-5 MB | ~150 MB | ~10 MB | ~50 MB |
| **内存占用** | 低 (系统 WebView) | 高 (自带 Chromium) | 中 | 高 |
| **启动速度** | <1秒 | 3-5秒 | <1秒 | 2-3秒 |
| **安全性** | 优秀 (Rust + CSP) | 一般 | 良好 | 一般 |
| **前端技术** | 任意 Web 技术 | 任意 Web 技术 | 任意 Web 技术 | 受限 (QML/Widgets) |
| **原生 API** | 丰富 (插件系统) | 丰富 (Node.js) | 基础 | 完整 Qt API |
| **自动更新** | 内置支持 | 需额外配置 | 需额外配置 | 需自行实现 |
| **开发体验** | 热更新、TypeScript | 热更新 | 热更新 | 重启调试 |

**Tauri 优势:**
- ✅ **极致轻量**: 编译后安装包仅 3-5MB（Electron 的 1/30）
- ✅ **原生性能**: 使用系统 WebView，内存占用低 80%
- ✅ **Rust 安全**: 后端逻辑用 Rust 编写，内存安全无崩溃
- ✅ **前端自由**: 可用 React/Vue/Svelte 等任意框架
- ✅ **跨平台**: Windows/macOS/Linux 一键构建
- ✅ **移动支持**: Tauri 2.0 支持 iOS/Android（未来扩展）

**Tauri 劣势:**
- ❌ 学习曲线（需要了解 Rust 基础）
- ❌ 生态相对 Electron 较小（但快速增长）
- ❌ WebView 兼容性依赖系统（Windows 需 WebView2）

---

### 2. shadcn/ui (UI 组件库)

**2025 年最受欢迎的 React 组件库**

**设计理念:**
- **不是传统组件库**: 组件代码直接复制到你的项目，完全可控
- **开放代码**: 可以任意修改组件实现，无黑盒
- **基于 Tailwind**: 样式通过 Tailwind CSS 实现，极易定制
- **Radix 底层**: 使用 Radix UI 提供无障碍访问支持

**核心优势:**

```
传统组件库 (MUI/AntD):
┌─────────────────┐      ┌──────────────┐
│   npm install   │ ───> │  黑盒组件    │
│   @mui/material │      │  难以定制    │
└─────────────────┘      └──────────────┘

shadcn/ui:
┌─────────────────┐      ┌──────────────┐
│ npx shadcn add  │ ───> │  源码组件    │
│    button       │      │  完全可控    │
└─────────────────┘      └──────────────┘
```

**包含组件 (50+):**
- 基础: Button, Input, Label, Badge, Card, Avatar
- 表单: Form, Select, Checkbox, Radio, Switch, Calendar
- 数据: Table, Data Table, Pagination, Tabs
- 反馈: Dialog, Alert, Toast, Progress, Skeleton
- 导航: Breadcrumb, Command (搜索), Menubar, Sidebar
- 布局: Resizable (分割面板), Scroll Area, Separator

## 设计方案特色

### 1. 视觉设计
- **设计语言**: 专业、稳重、高效的法律行业风格
- **配色方案**: 
  - 主色: 深蓝 (信任、专业)
  - 辅色: 金色/琥珀 (法律、品质)
  - 背景: 浅灰白 (护眼、长时间工作)
- **字体**: Inter (英文) + 思源黑体 (中文)
- **圆角**: 适度圆角 (8px)，既现代又稳重

### 2. 交互设计
- **流畅动画**: 页面切换、模态框、提示消息都有平滑过渡
- **键盘友好**: 支持快捷键操作（律师习惯键盘）
- **拖拽支持**: 文件拖拽识别、模板拖拽排序
- **实时预览**: 案卷生成前实时预览文件夹结构
- **暗黑模式**: 支持暗黑模式（夜间工作保护眼睛）

### 3. 功能增强
- **全局搜索**: Command + K 快速搜索模板、文件
- **最近使用**: 快速访问最近案卷和模板
- **批量操作**: 支持多文件批量 OCR 识别
- **数据可视化**: 案卷统计图表（案件类型分布、月度趋势）
