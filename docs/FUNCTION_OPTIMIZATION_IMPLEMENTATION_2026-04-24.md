# 现有功能优化落地记录（2026-04-24）

## 已落地

1. 归档中心步骤式流程

- 增加“选目录、选文件、抽取变量、选择模板、导出”步骤状态栏。
- 选择文件、设置/识别变量、选择模板、导出成功时自动推进状态。
- 案卷结构加载后启动后台扫描，右侧案卷结构显示完整文件数量。

2. 案件与文件操作异步化

- 归档文件树拖拽移动已在后台线程执行，并显示移动进度提示。
- 案件删除、批量剔除、案件文件夹迁移改为后台任务执行。
- 操作失败时增加回滚/核对说明，减少用户误判。

3. 模板系统诊断

- 模板管理新增“诊断”入口。
- 诊断内容包括失效模板路径、重复模板 ID、未定义 Word 占位符和占位符统计。
- 新增核心诊断模块，方便后续扩展为独立诊断页。

4. OCR/预览/文件树性能

- PDF 页渲染增加内存缓存，避免同一页在相同缩放/适应模式下重复渲染。
- OCR 截图识别增加内存缓存，相同截图重复识别时直接复用结果。
- 大目录继续使用懒加载，并增加后台完整扫描状态。

5. 测试与稳定性

- 新增时间提供器、平台路径适配器、文件系统适配器。
- 日历视图支持注入固定时间，减少日期相关测试脆弱性。
- 新增模板诊断与运行时适配器测试。
