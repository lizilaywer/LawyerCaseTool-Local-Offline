# 信息汇总文本框功能

## 功能概述

在信息识别（OCR）界面添加信息汇总文本框，自动将识别到的身份证信息按指定格式汇总显示，支持一键复制。

---

## 界面位置

位于识别结果字段区域下方，包含：
- 📋 **标题**："信息汇总"
- 📝 **文本框**：显示自动汇总的识别信息（可手动编辑）
- 📋 **一键复制按钮**：点击复制文本框内容到剪贴板

---

## 功能特性

### 1. 自动汇总

识别身份证（正面）后，自动按照以下格式汇总：

```
孙涛，男，1992年11月25日出生，汉族，住安徽省石台县丁香镇库山村查村组30号，公民身份号码：342922199211252514
```

### 2. 字段顺序

按照以下顺序组织信息：
1. **姓名** - 直接显示
2. **性别** - 格式：`，男/女`
3. **出生日期** - 格式：`，YYYY年MM月DD日出生`
4. **民族** - 格式：`，汉族`
5. **住址** - 格式：`，住XXX`
6. **身份证号** - 格式：`，公民身份号码：XXX。`（注意末尾有句号）

### 3. 手动编辑

文本框支持手动编辑，可以：
- 修改自动生成的汇总内容
- 添加额外信息
- 删除不需要的内容

### 4. 一键复制

点击"📋 一键复制"按钮：
- 将文本框内容复制到系统剪贴板
- 弹出提示"汇总信息已复制到剪贴板！"

## 使用示例

### 场景：识别身份证后复制信息

1. **添加身份证图片**并识别
2. **核对识别结果**字段
3. **查看汇总文本框**自动生成的格式文本
4. **手动修改**（如有需要）
5. **点击"一键复制"**按钮
6. **粘贴到**Word文档或其他地方
